# 🎮 大巴扎 - 战争策略游戏

一款基于HTML5 Canvas的塔防策略游戏，玩家需要购买各种战斗单位来抵御10波敌军的进攻。

## 🎯 游戏特色

- **10波塔防挑战**：每波敌军都有不同的兵种组合和难度
- **品质系统**：装备分为普通、优秀、稀有、史诗、传说、神话6个品质等级
- **丰富单位**：近战、远程、坦克、法师4大兵种，每种都有独特技能
- **策略布阵**：6格战斗区+6格背包，合理搭配是胜利的关键
- **移动端优化**：支持触摸操作，手机和平板都能流畅游玩

## 🕹️ 游戏玩法

### 基础操作
- 从商店购买单位放入战斗区
- 战斗区物品会自动生产单位参与战斗
- 拖拽调整战斗区中物品的位置
- 使用背包存储暂时用不上的物品

### 核心机制
- **金币系统**：初始12金币，每波结束获得5×波数的金币
- **品质升级**：购买相同物品可以提升品质和属性
- **协同效果**：相邻物品会产生各种协同加成
- **阶段切换**：备战阶段购买装备 → 战斗阶段自动战斗

### 单位介绍
- **近战系列**：⚔️战士、🥷忍者、🪓狂战士、👹野蛮人、💪巨人、🐎骑兵等
- **远程系列**：🏹战弓、🔮法杖、🛡️盾牌等
- **特殊物品**：🚩战旗、🗡️宝剑、🏅徽章等支援装备

## 🛠️ 技术栈

- **前端**：HTML5 Canvas、原生JavaScript
- **渲染**：自定义Canvas渲染引擎
- **特效**：粒子系统、动画效果
- **交互**：拖拽系统、触摸支持
- **图标**：Emoji字符集

## 🎲 游戏技巧

1. **前期节约**：第1-3波资源有限，优先购买性价比高的单位
2. **升级优先**：相同物品合成高品质比分散购买更有效
3. **位置重要**：相邻位置的协同效果是致胜关键
4. **兵种搭配**：近战+远程+坦克的组合较为均衡
5. **背包利用**：合理使用背包存储，等待合适时机组合
