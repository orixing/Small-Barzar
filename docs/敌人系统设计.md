# 敌人系统设计文档

## 概览

游戏共10波敌军，每波都有独特的兵种配置和主题。敌人系统包含出兵间隔、兵种配置、数量控制等机制。

## 波次配置

### 第1波 - 近战兵团 ⚔️
- **兵种配置**: 90%近战 + 10%坦克
- **主题**: 以近战单位为主的基础波次

### 第2波 - 弓箭军团 🏹
- **兵种配置**: 70%远程 + 20%近战 + 10%法师
- **主题**: 远程攻击为主

- **兵种配置**: 60%坦克 + 30%近战 + 10%远程

### 第4波 - 法师军团 🔮
- **兵种配置**: 60%法师 + 25%远程 + 15%近战
- **主题**: 魔法攻击为主

### 第5波 - 混合军团 ⚔️🏹
- **兵种配置**: 40%近战 + 40%远程 + 15%坦克 + 5%法师
- **主题**: 均衡的混合编队

### 第6波 - 精锐军团 🛡️⚔️
- **兵种配置**: 45%近战 + 45%坦克 + 10%远程

### 第7波 - 魔战军团 🔮⚔️
- **兵种配置**: 50%法师 + 35%近战 + 15%远程
- **主题**: 魔法与物理的结合

### 第8波 - 守护军团 🏹🛡️

## 出兵机制

### 出兵间隔

#### 固定间隔配置（秒）
```javascript
const intervalSeconds = {
    1: 3.5,   // 第1波: 3.5秒
    2: 3.0,   // 第2波: 3秒
    3: 2.5,   // 第3波: 2.5秒
    4: 2.0,   // 第4波: 2秒
    5: 3.0,   // 第5波: 3秒
    6: 2.5,   // 第6波: 2.5秒
    7: 3.5,   // 第7波: 3.5秒
    8: 4.5,   // 第8波: 4.5秒
    9: 4.0,   // 第9波: 4秒
    10: 3.5   // 第10波: 3.5秒
};
```

#### 动态间隔调整（已弃用）
原有的动态计算公式：`Math.max(90, 240 - wave * 15) + (wave >= 7 ? 48 : 0)`
- 基础间隔从240帧开始，每波递减15帧
- 第7波及以后额外增加48帧(0.8秒)
- 最小间隔限制为90帧(1.5秒)

### 每次出兵数量

```javascript
function getWaveSpawnCount() {
    if (wave >= 10) return 4;      // 第10波：每次4个
    else if (wave >= 8) return 3;  // 第8-9波：每次3个
    else if (wave >= 5) return 2;  // 第5-7波：每次2个
    else return 1;                 // 第1-4波：每次1个
}
```

### 单位数量限制

- **敌方单位上限**: 30个（防止性能问题）
- **玩方单位上限**: 50个（相对宽松）
- 达到上限时暂停生成新单位

## 敌方AI经济

### 初始金币
- **第1波**: 200金币

### 金币增长
- **每波开始**: +50 × 波数金币
- **每帧收入**: +1金币/帧
- **击杀玩家单位**: +单位成本金币
- **玩家单位到达基地**: +单位成本÷4金币

### 金币消耗
- 生产单位时扣除对应成本
- 金币不足时停止生产

## 兵种数据

### 兵种类型映射
- **melee**: 近战单位（剑士、战士等）
- **ranged**: 远程单位（弓箭手等）
- **tank**: 坦克单位（盾兵等）
- **mage**: 法师单位（法杖等）

### 单位属性
所有敌方单位保持基础数值，不会随波次成长，确保游戏平衡。

## 性能优化

### 单位管理
- 使用标记删除而非立即删除，减少数组操作
- 每5帧批量清理死亡单位
- 限制同时存在的单位数量

### 内存管理
- 战斗结束时清理所有单位引用
- 提供垃圾回收提示
- 避免内存泄漏

## 平衡性设计

### 难度曲线
- 第1-4波：递增难度，让玩家熟悉机制
- 第5-6波：中期挑战，测试玩家策略
- 第7-10波：后期高难度，需要完整的战略布局

### 兵种克制
- 不同波次的兵种配置鼓励玩家调整策略
- 避免单一流派通关，增加游戏深度

## 代码位置

主要逻辑位于 `js/game.js`：
- `spawnEnemyUnit()`: 敌人生成逻辑
- `getWaveUnitType()`: 兵种选择逻辑
- `getWaveSpawnCount()`: 数量控制
- `getWaveSpawnInterval()`: 间隔控制
- `updateAI()`: AI更新循环
