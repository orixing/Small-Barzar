# 游戏流程与机制

## 🎯 核心游戏循环

### 双阶段循环系统
游戏采用**备战阶段 → 战斗阶段**的双循环设计，每个循环对应一波敌人。

```
开始游戏 → 备战阶段 → 战斗阶段 → 结算 → 下一波备战阶段 → ...
```

## 📋 备战阶段 (Preparation Phase)

### 界面布局
- **敌人介绍区**：显示即将面对的敌人类型和特点
- **商店区域**：3个随机物品供购买
- **战斗区域**：6个装备槽位，可放置购买的物品
- **背包系统**：6个额外存储槽位

### 核心操作
1. **购买物品**：从商店购买装备到战斗区或背包
2. **物品管理**：拖拽移动物品位置，优化布局
3. **装备升级**：相同物品合并升级品质
4. **物品出售**：将不需要的物品卖回商店
5. **开始战斗**：点击"开始战斗"进入自动战斗

### 战略要素
- **位置摆放**：某些物品有相邻效果，需要考虑布局
- **资源管理**：平衡金币的消费和收益
- **装备组合**：不同装备的协同效应
- **商店刷新**：每波有限次数的免费刷新

## ⚔️ 战斗阶段 (Battle Phase)

### 自动战斗
- **完全自动化**：玩家无法直接控制，只能观战
- **物品触发**：装备按冷却时间自动生成单位
- **实时战斗**：单位自动移动、攻击、使用技能

### 胜利条件
- **玩家胜利**：摧毁敌方基地
- **玩家失败**：己方基地被摧毁

### 特殊机制
- **失败继续**：即使败北也可进入下一波（新机制）
- **技能触发**：某些装备的被动技能在战斗中生效
- **实时反馈**：战斗过程中的各种视觉特效

## 🎊 波次系统

### 10波设计
游戏共10波敌人，每波都有独特的主题和难度：

| 波数 | 主题 | 特点 |
|------|------|------|
| 1-2波 | 基础教学 | 单一兵种，节奏较慢 |
| 3-4波 | 难度递增 | 混合兵种，考验基础搭配 |
| 5-6波 | 中期挑战 | 策略要求提升 |
| 7-8波 | 高级挑战 | 复杂兵种组合 |
| 9-10波 | 终极考验 | 最高难度，全面考验 |

### 波次特色
- **敌人类型变化**：每波主打不同兵种组合
- **数量递增**：后期敌人数量和强度显著提升
- **节奏调整**：第7波后出兵间隔增加，给玩家更多思考时间

## 🏆 游戏结束条件

### 胜利条件
- **完成第10波**：成功击败所有10波敌人

### 失败条件
- **无**：新版本中移除了失败结束，失败后可继续下一波

### 结算奖励
- **金币奖励**：每波开始获得 5×波数 的金币
- **装备保留**：所有装备和升级效果会保持到下一波
- **技能重置**：部分临时效果（如狂战士击杀加成）会重置

## 🎮 操作流程

### 新手引导流程
1. 进入第1波备战阶段
2. 商店自动提供推荐装备组合
3. 购买并摆放装备
4. 点击开始战斗观看效果
5. 战斗结束后进入第2波准备

### 进阶策略流程
1. 分析即将面对的敌人类型
2. 根据敌人特点调整装备搭配
3. 合理利用相邻效果优化布局
4. 管理资源，决定升级还是扩展
5. 观察战斗结果，为下波做准备

## 📊 核心数值

### 时间节奏
- **备战阶段**：无时间限制，玩家自主节奏
- **战斗阶段**：通常1-3分钟，取决于双方实力

### 资源系统
- **初始金币**：12
- **波次奖励**：5×波数
- **商店刷新**：每波限制次数的免费刷新

### 难度曲线
- **敌人血量**：50 + 50×波数
- **玩家基地血量**：同敌人，每波重置
- **敌人出兵间隔**：根据波数动态调整
